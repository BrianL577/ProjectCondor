Data integrity policy

Every data point in every report is labeled either ✓ verified (traceable to a named public source) or ! check source (requires manual confirmation before use in deal materials). Competitor names are never auto-populated unless confirmed from the FAA Part 145 database or UK CAA register. This policy is enforced in the AI prompt and cannot be overridden.

About this system
How the autonomous report system works.
Architecture

Every Monday at 6 AM ET, a GitHub Actions scheduler automatically runs a script that calls Google's Gemini AI for all 22 locations. Each report is saved to a Supabase database. This website reads from that database and shows you the latest reports — no manual action needed, ever.
